Descripción del sistema

Este proyecto desarrolla un sistema básico de gestión de restaurante utilizando Programación Orientada a Objetos en Python. El sistema permite registrar productos y clientes, almacenarlos dentro de un restaurante y mostrar la información registrada en consola.

Estructura del proyecto

restaurante_app/

modelos/
producto.py
cliente.py
servicios/
restaurante.py
main.py
Explicación de la estructura

La carpeta modelos contiene las clases que representan las entidades principales del sistema: Producto y Cliente.

La carpeta servicios contiene la clase Restaurante, encargada de gestionar los productos y clientes registrados.

El archivo main.py es el punto de inicio del programa, donde se crean los objetos y se ejecutan los métodos para demostrar el funcionamiento del sistema.
